# pxlpong
Simple Pong game I put together to learn Love2d.
![screenshot](/screenshot.png)

## controls
Up/Down or Mouse: move player 1 paddle  
A/Z: move player 2 paddle  
Enter: start game  
Arrow keys for menu config
